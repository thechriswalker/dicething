# EDDC - Everyday Dice Carry

Boxes based on: https://www.printables.com/model/116056-every-day-dice-carry-eddc-box-blanc by https://www.printables.com/@wuerfel/collections

The idea here is to add a "box builder" sectoin of the app.

Import a dice set and have a box built to fit them all perfectly.
Based on this box but with configurable magnet size (optional print-in magents) and housings to hold each die by shape.
